# 🏫 College Automation Software
A robust desktop application built in Python to streamline college administrative workflows, including student records, attendance tracking, and performance analytics.

### 🚀 Key Features
- **Secure Authentication:** Multi-level login system for administrators and students.
- **Database Management:** Full CRUD (Create, Read, Update, Delete) functionality for student profiles using **SQLite**.
- **Performance Analytics:** Automated calculation of CGPA/attendance percentages with report generation.
- **Modern GUI:** Clean and intuitive interface designed using the **Tkinter** library.

### 🛠 Tech Stack
- **Language:** Python 3.x
- **GUI Framework:** Tkinter
- **Database:** SQLite (or MySQL)
- **Libraries:** Pandas (for report generation), Matplotlib (for performance charts)
